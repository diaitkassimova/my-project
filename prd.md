# ТЗ: «Учим Алфавит вместе» — тренажёр русского алфавита для дошкольников

## 1. Цель продукта

Одностраничный веб-сайт для изучения русского алфавита ребёнком 6 лет. Ребёнок видит букву, запоминает её написание, а затем видит картинку предмета/животного, название которого начинается на эту букву (для Ъ, Ы, Ь — буква встречается внутри слова). Буквы выдаются в случайном порядке.

## 2. Целевая аудитория

Дети 5–7 лет, дошкольный возраст, самостоятельное использование или с родителем. Требуется минимальная грамотность взаимодействия: один основной элемент управления на экране, крупные кнопки, отсутствие текстовых инструкций, которые ребёнок должен читать сам.

## 3. Пользовательский сценарий (основной флоу)

1. **Стартовый экран**: приветствие, маскот, кнопка «Начать».
2. **Экран буквы**: крупно показывается буква (заглавная + строчная), вокруг — кольцевой таймер на 5 секунд. Буква озвучивается голосом.
3. **Экран слова**: через 5 секунд автоматически показывается эмодзи-картинка и слово, в котором целевая буква подсвечена. Слово озвучивается голосом. Анимация звёздочек/конфетти.
4. Через ~4.5 секунды автоматически выбирается следующая случайная буква (без повтора, пока не пройдены все 33).
5. После прохождения всех 33 букв — экран поздравления с возможностью начать заново.

Ребёнок в любой момент может вручную нажать «Дальше» (пропустить ожидание), «Повторить» (повторить текущий шаг) или «Остановить обучение» (прервать сессию и вернуться на стартовый экран).

## 4. Функциональные требования

| № | Требование |
|---|---|
| F1 | Показ всех 33 букв русского алфавита (А–Я, включая Ё, Й, Ъ, Ы, Ь). |
| F2 | Для каждой буквы — одно слово-пример и одна картинка (эмодзи), начинающиеся на эту букву. |
| F3 | Для Ъ, Ы, Ь — специальная подача: буква подсвечивается внутри слова + короткая понятная подсказка о её роли. |
| F4 | Случайный порядок выдачи букв без повторов до исчерпания полного набора; после прохождения всех — новая перетасовка. |
| F5 | Таймер показа буквы — 5 секунд (визуальный кольцевой индикатор). |
| F6 | Автоматический переход буква → слово → следующая буква без обязательного участия ребёнка. |
| F7 | Голосовое озвучивание буквы и слова (Web Speech API, `ru-RU`), с автоматическим подбором наиболее реалистичного/натурального доступного русского голоса (не робо-голос по умолчанию), с щадящей скоростью и тоном произношения для восприятия ребёнком. |
| F8 | Кнопка включения/выключения звука. |
| F9 | Кнопка «Повторить» — повтор текущего шага без перехода дальше. |
| F10 | Кнопка «Дальше» — ручной пропуск ожидания. |
| F11 | Индикатор прогресса — полоса из 33 ячеек, отражающая пройденные и текущую буквы. |
| F12 | Финальный экран по завершении всего алфавита с возможностью перезапуска. |
| F13 | Кнопка «Остановить обучение» — доступна на экранах буквы и слова, немедленно прерывает текущую сессию (останавливает таймеры и озвучку) и возвращает пользователя на стартовый экран без сохранения прогресса. |

## 5. Нефункциональные требования

- **Дружелюбный визуальный стиль**: яркая пастельная палитра, скруглённые формы, декоративные плавающие элементы (облака, звёзды, шары), крупный «мультяшный» шрифт (Comfortaa) для букв и заголовков.
- **Одностраничность**: вся логика в одном HTML-файле, без серверной части и без внешних API (кроме шрифтов Google Fonts).
- **Офлайн-совместимость контента**: картинки — эмодзи, а не внешние изображения, чтобы сайт работал без загрузки медиа с сервера.
- **Адаптивность**: корректное отображение на десктопе и планшете/телефоне (в т.ч. по высоте экрана).
- **Доступность управления**: крупные кликабельные/тапабельные кнопки, минимум текста для чтения.
- **Не требует установки/логина**: открывается как обычный HTML-файл в браузере.

## 6. Контентная модель

Для каждой из 33 букв определены поля:

- `l` — буква (заглавная);
- `w` — слово-пример;
- `e` — эмодзи-иллюстрация;
- `note` (опционально) — пояснение для Ъ, Ы, Ь.

Список слов-примеров: Арбуз, Барабан, Волк, Гриб, Дом, Ель, Ёжик, Жираф, Заяц, Индюк, Йогурт, Кот, Лев, Мышь, Носорог, Обезьяна, Пингвин, Рыба, Слон, Тигр, Утка, Фламинго, Хомяк, Цыплёнок, Черепаха, Шар, Щенок, Подъезд (Ъ), Мыло (Ы), Конь (Ь), Экскаватор, Юла, Яблоко.

## 7. Технический стек

- HTML5 + CSS3 + Vanilla JavaScript (без фреймворков и сборки).
- Web Speech API (`SpeechSynthesisUtterance`, `lang: ru-RU`) для озвучивания.
- SVG для анимации кольцевого таймера.
- Google Fonts (Comfortaa, Nunito) — единственная внешняя зависимость.

## 8. Экраны и состояния (state machine)

`start → letter → word → letter → word → … → done → (restart) → start`

Кнопка «Остановить обучение» доступна из состояний `letter` и `word` и переводит машину состояний напрямую в `start`, минуя `done`.

Переходы управляются таймерами (`setTimeout`), которые сбрасываются при ручном вмешательстве (кнопки «Дальше»/«Повторить»/«Остановить обучение»), чтобы избежать наложения анимаций и повторной озвучки.

## 9. Критерии приёмки

- Все 33 буквы алфавита воспроизводятся в правильном написании, без пропусков и дублей за один проход.
- Ъ, Ы, Ь не подаются как «начинающиеся» буквы — их появление сопровождается пояснением.
- Таймер показа буквы визуально отсчитывает ровно 5 секунд перед переходом к картинке.
- Порядок букв случайный при каждом запуске/перезапуске.
- Сайт открывается и полностью работает как один HTML-файл без интернета (за исключением загрузки шрифтов и, при наличии, голосов озвучки).
- Управление доступно и понятно ребёнку 6 лет без чтения инструкций (звук, повтор, далее, остановить — как единственные постоянные элементы).
- Голос озвучивания звучит естественно (не механически): при наличии в системе нескольких `ru-RU` голосов выбирается наиболее качественный/натуральный из доступных.
- Нажатие «Остановить обучение» мгновенно прекращает текущую озвучку и таймеры и возвращает на стартовый экран.

## 10. Вне рамок (Out of scope)

- Отслеживание прогресса между сессиями (localStorage/аккаунты).
- Мультиязычность.
- Игровые режимы (тест на знание букв, счёт очков).
- Серверная часть, аналитика, монетизация.
